# cmx — CleanMyMac X CLI Equivalent

```
cmx status               System overview (disk/memory/load)
cmx scan                 Full-module smart scan
cmx clean system-junk    Clean system junk (dry-run by default)
cmx optimize [--apply]   Performance optimization
cmx uninstall <app>      Uninstall app + leftover files
```

**Dry-run by default** (preview only). Add `--apply` to execute.

See `cmx help` or the `SKILL.md` in the parent directory.
